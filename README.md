# Реактивный WebFlux-сервис «Калькулятор»

## Общее описание

Реализован реактивный RESTful WebFlux-сервис «Калькулятор», выполняющий периодические вычисления двух произвольных пользовательских функций, заданных в текстовом виде на JavaScript (либо Python), и транслирующий результаты в виде потокового CSV-ответа.

## Конфигурационный файл

Конфигурационные параметры сервиса инициализируются на основании внешнего JSON-документа, подгружаемого при старте приложения.

**Пример содержимого `config.json`:**

```json
{
    "function1": "function(x) { return x * x; }",
    "function2": "function(x) { return x + 10; }",
    "interval": 10
}
